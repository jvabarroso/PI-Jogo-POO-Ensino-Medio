MAGO MESTRE
📜 Bem-vindo à Tribo da Sobrecarga, jovem mago!
Sobrecarga é quando um mesmo método tem versões diferentes, mudando os parâmetros. Exemplo: lançar poder() com 1, 2 ou 3 cristais – cada um causa um efeito distinto.
// Exemplo:
poder(int cristal) → "🔥 Chama"
poder(int cristal, String tipo) → "⚡ Raio elemental"
poder() → "🔮 Magia arcana"
✨ Seu objetivo: complete as lacunas do código para lançar a magia com 2 cristais. Vamos praticar?
